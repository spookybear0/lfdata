# lfdata

A Python project for managing LF data, sourced from TDF files.

Note that this codebase is partially autogenerated, so it's hella ugly and
prone to large-scale changes.

If you want to see nicer, artisanal hand-crafted organic lf-based code,
check out https://github.com/spookybear0/laserforce_ranking and
https://github.com/zmaniacz/lfstats-next.
